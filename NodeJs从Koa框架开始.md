### Vscode快捷键
- 重开一行：光标在行尾的话，回车即可；不在行尾，cmd + enter 向下重开一行；shift + cmd + enter 则是在上一行重开一行
- 删除一行：光标没有选择内容时，cmd + x 剪切一行；shift + cmd + k 直接删除一行
- 移动一行：alt + ↑ 向上移动一行；alt + ↓ 向下移动一行
- 复制一行：shift + alt + ↓ 向下复制一行；shift + alt + ↑ 向上复制一行
