1.在ES6中，如果一个函数只接收一个形参且函数提示一个return语句，使用箭头函数简化
```js
const fun1 = p => v;
// 类似es5
const fun2 = function(p) {
    return v
}
```

2.多层嵌套时，可以简化
```js
// es6
const func1 = a => b => c => a + b +c
// es5
const func2 = function(a) {
    return function (b) {
        return function (c) {
            return a + b + c;
        }
    }
}
```
