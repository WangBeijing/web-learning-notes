#### 适配器模式
**介绍**
- 旧接口格式和使用者不兼容
- 中间加一个适配转换接口

**UML类图**
![image]()
**代码演示**
```js
class Adaptee {
    specificRequest() {
        return '德国标准插头'
    }
}
class Target {
    constructor() {
        this.adaptee = new Apatee()
    }
    request() {
        let info = this.adaptee.specificRequest()
        return `${info} - 转换器 - 中国标准插头`
    }
}
//测试
let target = new Target()
let res = target.request()
console.log(res)
```
**适配器应用场景**
- 封装旧接口
- vue computed

**1.封装旧接口**
```js
//自己封装的ajax，使用方式如下：
ajax({
    url: '/getData',
    type:'Post',
    dataType:'json',
    data:{
        id:'123'
    }
}).done(function(){})

// 但因为历史原因，代码中全是：
$.ajax({...})
```
```js
//做一层适配
var $ = {
    ajax: function(option){
        return ajax(option)
    }
}
```
